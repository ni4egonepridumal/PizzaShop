Магази пицц

Написан на реакт

Перед запуском проекта в корневой папке в терминале запустит команду 

"npx json-server --watch public/db.json --port 3001"

для получения бекенда, и загрузки пицц

затем запускать проект в новом терминале

Использовал:

ReactJS + хуки

React Router

Redux

Redux thunk

json-server

Axios

без адаптива.
